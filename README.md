# yt-dlp バッチファイル集 yt-dlp-bat-pack

> **動画のURLを貼り付けるだけ。**  
> コマンドプロンプトに不慣れな方でも、YouTube・ニコニコ動画など、さまざまな動画サイトから簡単にダウンロードできる Windows 用バッチファイル集です。

---

## できること

- **MP4 / MKV** — 動画のダウンロード（画質選択対応）
- **MP3 / AAC / M4A / FLAC / OPUS / WAV** — 音声のみのダウンロード
- **プレイリスト一括** — 再生リストの動画をまとめてダウンロード
- **統合ダウンローダー** — 1つの画面から全フォーマット・画質・音質を選択可能

---

## 必要な環境

本バッチファイルの実行には、以下のプログラムが **PATH に登録されている** 必要があります。

| プログラム | 用途 | 入手先 |
|:--|:--|:--|
| `yt-dlp.exe` | 動画のダウンロード | [yt-dlp/yt-dlp](https://github.com/yt-dlp/yt-dlp/releases) |
| `ffmpeg.exe` | 動画・音声の変換・結合 | [ffmpeg.org](https://ffmpeg.org/download.html) |
| `ffprobe.exe` | メディア情報の解析 | ffmpeg に同梱 |

ダウンロード方法はnoteにて解説しています

(https://note.com/eigenarc/n/n4be318a68c0e)

[**yt-dlpの環境構築ガイド｜Windows11へのインストール方法を完全解説**]
(https://note.com/eigenarc/n/ndfd77945aa21)

## 使い方

### 基本的な流れ

1. 使いたいバッチファイル(.bat)を **ダブルクリック** して起動
2. コマンドプロンプトが開き、URLの入力を求められる
3. ダウンロードしたい動画の **URLを貼り付け**（右クリック or Ctrl+V）して `Enter`
4. 形式や画質の選択画面が表示される場合は、番号を入力して `Enter`
5. ダウンロード完了後、`downloads` フォルダにファイルが保存される

詳しくはnoteでも解説しています。
[**yt-dlp用batファイルの無料配布と使い方**]

# 【重要】初回起動時の「セキュリティの警告」について

ダウンロードしたバッチファイル（.bat）を実行する際、Windowsのシステムによって**「開いているファイル - セキュリティの警告」**という画面が表示され、「発行元を確認できませんでした」というメッセージが出ることがあります。

### ■ なぜこの警告が出るの？
この表示はウイルスや不正なプログラムを検知したという警告**ではありません**。

企業などが有料で取得する「デジタル署名」が付与されていない、個人制作のプログラムやスクリプトをWindows上で実行しようとした際、OSの仕様として自動的に表示される標準的な確認画面です。中身は安全なコードですのでご安心ください。

### ■ 起動の手順
1. 警告画面の右下にある **「実行(R)」** ボタンをクリックしてください。
2. ツールが正常に起動します。

> ** ヒント**
> 左下にある「このファイルを開く前に常に確認する(W)」のチェックを外してから「実行(R)」を押すと、次回以降はこの確認画面が表示されなくなります。

---

### ■ セキュリティ面で不安を感じる方へ
バッチファイルは透明性の高いプログラムです。実行前にファイルを右クリックして「編集（またはメモ帳で開く）」を選択すれば、**すべての命令コードを直接確認することができます。**

どうしても本ファイルから直接実行することに抵抗がある場合は、以下の手順で**ご自身による「自作バッチ」**として実行していただくことも可能です。

1. 本ツールで公開しているソースコードをすべてコピーする。
2. 新規のテキストドキュメント（メモ帳など）を作成し、コードを貼り付ける。
3. ファイル名を任意のものに変更し、拡張子を **`.bat`** に書き換えて保存する。

ご自身で作成したファイルであれば、Windowsからの警告は発生しません。利用環境やポリシーに合わせて、安心できる方法でご活用ください。

## 配置場所について

本バッチファイルは **任意の場所** に配置して使用できます。  
ダウンロードされたファイルは、バッチファイルと同じ場所に自動生成される `downloads` フォルダ内に形式ごとに保存されます。

> [!WARNING]
> フォルダパスに特殊文字や一部の日本語が含まれていると、エラーが発生する場合があります。  
> うまく動かない場合は、シンプルなパス（例: `C:\yt-dlp\`）に配置してください。

## ❓ よくあるトラブル

<details>
<summary><strong>バッチファイルを開いても一瞬で画面が消える</strong></summary>

`yt-dlp` や `ffmpeg` が見つからない（PATH が通っていない）ことが原因です。  
コマンドプロンプトで以下を実行して、正常に表示されるか確認してください。

```
yt-dlp --version
ffmpeg -version
```

</details>

<details>
<summary><strong>ダウンロードが開始されない / 途中で止まる</strong></summary>

動画サイト側の仕様変更により、現在のバージョンの yt-dlp が対応できなくなっている可能性があります。  
**`yt-dlpアップデート.bat`** を実行して最新版に更新してから再度お試しください。

> 会員限定動画や DRM（デジタル著作権管理）で保護されたコンテンツはダウンロードできません。

</details>

<details>
<summary><strong>ダウンロードした動画の音声が出ない</strong></summary>

MP4 形式でダウンロードした場合に発生することがあります。  
VP9 映像や Opus 音声が MP4 コンテナに格納されると、Windows 標準プレイヤーで音声が再生されない場合があります。  
本バッチファイル集では、この問題を回避するフォーマット指定がされていますが、万が一発生した場合は **MKV 形式** でのダウンロードをお試しください。

</details>
